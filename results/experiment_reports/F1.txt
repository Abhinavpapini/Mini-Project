MODEL CONFIGURATION
=====================================================================

Feature input:
  - HuBERT-large layer 21 (1024-dim → PCA-32, expl. variance 58.3%)
  - MFCC-21 (first 21 coefficients from 78-dim stats cache, standardised)
  - Fused dim: 53  (32 + 21)

Model architecture: CNN-1D multi-label
  - Input reshaped to [B, 1, 53]   (1 channel, 53-length sequence)
  - Conv1d(1→64, k=7) + BN + GELU
  - Conv1d(64→128, k=5) + BN + GELU
  - Conv1d(128→256, k=3) + BN + GELU
  - AdaptiveAvgPool1d(1) → Flatten → [B, 256]
  - Dropout(0.3) → Linear(256→128) → GELU
  - Dropout(0.3) → Linear(128→5)  → (sigmoid at inference)
  - Total params: 174,597

Loss: Focal Loss (gamma=2.0, per-class alpha)
  Focal alpha values (inv-frequency weighted, train fold only):
    Block        : 0.5755
    Prolongation : 0.6970
    SoundRep     : 0.7895
    WordRep      : 0.8171
    Interjection : 0.6415

Sampling: WeightedRandomSampler
  - Sample weight = max(inverse class freq) across positive labels
  - Rare stutter clips (WordRep, SoundRep) upsampled automatically

Training:
  - Optimizer  : AdamW (lr=3e-4, wd=1e-4)
  - Scheduler  : CosineAnnealingLR (30 epochs, eta_min=1.5e-5)
  - Batch size : 256
  - Threshold  : 0.5 sigmoid decision boundary
  - Train/test : 80/20 stratified on Block column, seed=42

=====================================================================
KEY INTERPRETATIONS
=====================================================================

1. HIGH RECALL, LOW PRECISION: All classes show recall >> precision (avg 0.73 vs 0.46).
   The focal loss + class-balanced sampler is aggressively pushing the model to detect
   positives (good for screening) at the cost of false positives. This is directionally
   correct for clinical use but would cause annotation fatigue.

2. WORD REPETITION REMAINS HARDEST (F1=0.415): Even with focal loss + oversampling,
   WordRep precision=0.285 is very low. The model is essentially guessing positive for
   ambiguous clips. WordRep needs richer temporal features (full sequence, not pooled).

3. INTERJECTION REMAINS EASIEST (F1=0.642): Consistent with C5. Lexically stereotyped
   fillers are learnable even in the multi-label setting.

4. LATE BEST EPOCH (28/30): Unlike C5 which peaked at epoch 4-17, F1 continues
   improving through epoch 28. Worth retrying with 50 epochs to see if macro-F1
   breaks 0.56+.

5. AUPRC (0.581 macro): Ranking quality is reasonable given the class imbalance.
   WordRep has lowest AUPRC (0.460) — the model's confidence for this class is
   poorly calibrated.

6. TRAINING LOSS STEADILY DECREASING (0.0704 → 0.0596): No overfitting detected —
   val_loss also decreases throughout. The capacity (174K params) is not too large.

7. PCA-32 BOTTLENECK: Only 58.3% of HuBERT variance retained; combined with the
   tiny MFCC-21 input, 53 total dims may not capture enough structure. F4 will
   test a deeper model architecture on this same input.

Key Hypotheses and Expected Outcomes
------------------------------------
- H1: Layer optimality is stutter-type dependent: Blocks (motor planning failure)
  may be better captured in later layers encoding prosodic structure, while sound
  repetitions (acoustic-phonetic events) may peak in mid-layers. Expected: per-type
  optimal layers differ by 3 to 8 layers.
- H2: Dimensionality reduction improves generalization: PCA to about 32 dimensions
  from 1024 should improve macro-F1 on minority stutter types by removing noise
  dimensions, similar to the 32-dim result in the dysarthria paper.
- H3: Handcrafted stutter features provide complementary signal: Features like
  jitter, pause ratio, and speaking rate should show low correlation with compact
  SSL+MFCC fusion (similar to <0.1 mean correlation in the dysarthria paper),
  indicating additive value.
- H4: Graph-based models outperform CNN on temporal stutter patterns: Disfluencies
  span multiple frames and involve non-local acoustic dependencies (for example,
  coarticulation changes before a block). GCN/D-GAT are expected to capture these
  better than frame-averaged CNN classifiers.
- H5: Multi-label formulation improves co-occurrence handling: Converting to
  multi-label binary classification should improve F1 on clips with two or more
  simultaneous disfluency annotations, expected to be a significant portion of
  SEP-28k.
