=====================================================================
MODEL CONFIGURATION
=====================================================================

Feature input:
  - HuBERT-large layer 21 (1024-dim → PCA-32)
  - Explained variance: 58.3%

Embedding network (same CNN backbone as F1):
  Conv1d(1→64, k=7) + BN + GELU
  Conv1d(64→128, k=5) + BN + GELU
  Conv1d(128→256, k=3) + BN + GELU → AdaptiveAvgPool(1)
  Flatten → Dropout(0.2) → Linear(256→128) → LayerNorm(128)
  Total params: 174,208

Episodic training:
  - Episodes: 2000  (no "epochs" — pure episodic)
  - N-way: 5  (all stutter types)
  - K-shot: 8  (8 support examples per class per episode)
  - Queries: 16 per class per episode
  - Optimizer: Adam (lr=1e-3), StepLR(step=500, γ=0.5)

Prototype computation (at inference):
  For each stutter type t:
    proto_pos = mean(train_emb[y_t == 1])   [D]-dim centroid
    proto_neg = mean(train_emb[y_t == 0])   [D]-dim centroid
  Score = d(x, proto_neg) - d(x, proto_pos)   (positive = closer to pos)
  Threshold: 0.5 on min-max normalised score

=====================================================================
TRAINING DYNAMICS
=====================================================================

Episodes 1-500: Steady improvement (0.22 → 0.28 episode accuracy)
Episodes 500-900: Rapid improvement to 0.307 (learning meaningful features)
Episodes 900-2000: Plateau at 0.29-0.32 (embedding converged)

Best episode accuracy = 0.316 @ ep1700.
Note: 5-way random chance = 0.200. Achieved = 0.316 (+57.8% over chance).

The plateau from ep900 onwards is expected — with PCA-32 input, the embedding
space has limited degrees of freedom. The model has found a local optimum where
stutter types are separated in 128-dim space but substantial class overlap remains.

Episode accuracy of 0.316 for 5-way 5-class classification across all stutter
types (which co-occur heavily) is non-trivial — the model is learning useful
class boundaries despite 20-42% positive rates that create multi-label overlap.

=====================================================================
KEY INTERPRETATIONS
=====================================================================

1. D8 IS 3RD BEST MULTI-LABEL MODEL (Macro-F1=0.514):
   Despite no explicit class weighting, no focal loss, and no BCE loss — purely
   episodic prototype learning achieves competitive results. This confirms that
   episodic sampling is an effective implicit rebalancing strategy.

2. BEST MINORITY CLASS RECALL (WordRep=0.665, SoundRep=0.660):
   Prototypical training forces equal K-shot sampling from all classes each
   episode, preventing the model from ignoring rare classes. The embedding
   space captures minority class structure better than cross-entropy approaches.

3. HIGH RECALL / LOWER PRECISION TRADE-OFF:
   The prototype score (distance ratio) tends toward positive predictions —
   Macro-Recall=0.638 vs F1=0.726 (focal, highest recall) and D1=0.526.
   D8 is closer to F1 than D1 in terms of recall/precision balance.

4. EPISODE ACCURACY (0.316) IS MEANINGFUL BUT LIMITED:
   The 5 stutter types are highly overlapping classes (a clip can be both
   Block AND Prolongation). Treating episodes as 5-exclusive classes is
   an approximation — the true label space is multi-hot. Episode accuracy
   being interpreted as "which single class" underestimates model quality.

5. COULD BENEFIT FROM MORE EPISODES:
   The loss still decreasing at ep2000 (1.532 vs 1.613 at ep100) and accuracy
   still has slight upward trend. Running 5000 episodes might improve results.

6. EMBEDDING SPACE STRUCTURES:
   With 174K params (same topology as F1's 174K), D8 produces a metric space
   where t-SNE/UMAP would show stutter type clustering. This is unique value
   not available from D1/F1 (discriminative classifiers don't produce
   interpretable distance structure).
