Objective
---------
Ensemble the two strongest GPU-trained checkpoints (C4 and D7) without any
retraining. Pure inference-time combination by weighted averaging of sigmoid
probabilities, followed by a threshold sweep to find the optimal decision boundary.

This was motivated by G1's finding that retraining with changed hyperparameters
hurt overall performance. The solution: keep the original checkpoints as-is and
simply combine their predictions.

Data and Setup
--------------
Dataset        : SEP-28k + FluencyBank (labels)
Features       : HuBERT-large Layer 21 (1024-dim) + Whisper-large Layer 28 (1280-dim)
                 Pre-cached -- no GPU inference needed at runtime
Test Split     : 20% stratified (seed=42, identical to C4/D7 original splits)
Test Samples   : 4,182

Label Distribution (test set, positive rate):
  Block          : 42.4%
  Prolongation   : 30.5%
  SoundRep       : 21.2%
  WordRep        : 18.2%
  Interjection   : 36.0%

Ensemble Strategy
-----------------
Method    : Weighted sigmoid probability average
  C4 weight : 0.55  (given higher weight as strongest single model)
  D7 weight : 0.45
Threshold : Swept over [0.38, 0.40, 0.42, 0.44, 0.45, 0.46, 0.48, 0.50, 0.52]
Hardware  : CPU (inference only -- no training required)
Runtime   : ~9 seconds total

Key Findings
------------
1. ENSEMBLE BEATS BOTH INDIVIDUALS: G2 (0.6753) > C4 (0.6587) > D7 (0.6493).
   The two models make complementary errors -- D7 is better at Prolongation
   (0.6122 vs 0.5887) while C4 is better at WordRep (0.6949 vs 0.6882).
   Their combination lifts all classes except WordRep marginally.

2. LOWER THRESHOLD IS KEY: Best F1 found at threshold=0.40, not the default 0.50.
   Ensemble probabilities are naturally lower (averaging two models both below 1.0),
   so a lower threshold is needed to recover the same recall as individual models.
   This is a known property of ensemble averaging.

3. RECALL IMPROVED SIGNIFICANTLY: G2 Macro-Rec=0.6968 vs C4 Macro-Rec=0.6087.
   Ensemble catches more actual stutter events -- especially Block (+0.12) and
   Prolongation (+0.15). This is clinically valuable (missing stutters = worse than
   false alarms in a screening context).

4. AUPRC ALSO IMPROVED: G2 AUPRC=0.7479 vs C4 AUPRC=0.7459. The ensemble
   produces better-calibrated probability rankings, making it more useful for
   ranked retrieval or confidence-based filtering.

5. ZERO COST: G2 required no retraining, no GPU, no hyperparameter tuning.
   Total runtime was ~9 seconds on CPU. This is the most efficient improvement
   in the entire project -- +1.66% F1 for free.

6. WORDREP SLIGHT DROP: WordRep went from 0.6949 (C4) to 0.6895 (G2, -0.005).
   D7's WordRep (0.6882) is slightly weaker than C4, pulling the ensemble down
   marginally. Still above the SVM baseline of 0.6456.
