=====================================================================
WHY MDS UNDERPERFORMS LINEAR PCA
=====================================================================

1. HIGH MDS STRESS (19.8M) — POOR DISTANCE PRESERVATION:
   The raw stress value is very high. This means 3000 points in 128-dim
   cannot be faithfully embedded into 32-dim while preserving all pairwise
   distances. The 32-dim MDS solution is a heavily distorted representation.
   PCA-32 (which retains 72.9% variance from PCA-128) captures more useful
   variation than the distorted MDS distance-map.

2. RIDGE REGRESSION EXTENSION ADDS NOISE:
   The out-of-sample extension (landmark MDS → Ridge → full dataset) introduces
   approximation error. The Ridge model can only linearly extrapolate the MDS
   coordinates. This linear mapping from 128-dim → MDS-32 is essentially
   a different linear projection — weaker than PCA's guaranteed optimality.

3. TRAINING INSTABILITY (best F1 at ep17, oscillation 0.19-0.467):
   Similar to B2 — MDS features are harder for CNN to learn from than PCA.
   The non-monotonic training (ep5=0.19, ep7=0.447, ep10=0.39) suggests the
   MDS embedding lacks the smooth structure that linear PCA provides.

4. DISTANCE PRESERVATION ≠ CLASS DISCRIMINATION:
   MDS preserves pairwise distances (geometric fidelity) but this doesn't
   imply class separability. Linear PCA retains the directions of maximum
   variance which happen to contain the most inter-class variation for
   HuBERT features.

=====================================================================
B-SERIES KEY FINDING (B1 + B2 + B3)
=====================================================================

⭐ FINDING: For deep SSL features (HuBERT-large), linear PCA is the optimal
   dimensionality reduction method. Neither kernel-based (B2) nor
   distance-based (B3) non-linear reductions improve performance.

   Explanation: HuBERT's 24 transformer layers have already performed the
   necessary non-linear feature transformation. The resulting feature manifold
   is approximately linear at the macro level — PCA efficiently extracts the
   discriminative directions. Additional non-linear compression destroys this
   carefully learned structure.

   Practical guideline: Use simple linear PCA-32 to 64 with HuBERT features.
   Avoid KPCA (destroys kernel structure) and MDS (poor stress, noisy extension).

Key Hypotheses and Expected Outcomes
------------------------------------
- H1: Layer optimality is stutter-type dependent: Blocks (motor planning failure)
   may be better captured in later layers encoding prosodic structure, while sound
   repetitions (acoustic-phonetic events) may peak in mid-layers. Expected: per-type
   optimal layers differ by 3 to 8 layers.
- H2: Dimensionality reduction improves generalization: PCA to about 32 dimensions
   from 1024 should improve macro-F1 on minority stutter types by removing noise
   dimensions, similar to the 32-dim result in the dysarthria paper.
- H3: Handcrafted stutter features provide complementary signal: Features like
   jitter, pause ratio, and speaking rate should show low correlation with compact
   SSL+MFCC fusion (similar to <0.1 mean correlation in the dysarthria paper),
   indicating additive value.
- H4: Graph-based models outperform CNN on temporal stutter patterns: Disfluencies
   span multiple frames and involve non-local acoustic dependencies (for example,
   coarticulation changes before a block). GCN/D-GAT are expected to capture these
   better than frame-averaged CNN classifiers.
- H5: Multi-label formulation improves co-occurrence handling: Converting to
   multi-label binary classification should improve F1 on clips with two or more
   simultaneous disfluency annotations, expected to be a significant portion of
   SEP-28k.
