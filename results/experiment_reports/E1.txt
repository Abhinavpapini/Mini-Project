=====================================================================
MODEL CONFIGURATION
=====================================================================

Feature input:
  - HuBERT-large layer 21 (1024-dim → PCA-32, fit on train → applied to all)
  - All 20906 clips served as graph nodes (transductive)

Graph construction:
  - Cosine similarity between PCA-32 vectors
  - k=10 nearest neighbours per node (symmetric — if A→B, then B→A)
  - 162,328 directed edges → 81,164 undirected edges + 20,906 self-loops
  - A_hat = D^(-1/2) (A+I) D^(-1/2)  (symmetric normalised)

Architecture: 2-layer GCN
  Layer 1: GCNLayer(32→64) + BatchNorm1d(64) + ReLU + Dropout(0.3)
  Layer 2: GCNLayer(64→32) + Dropout(0.3)
  Head   : Linear(32→64) → GELU → Dropout → Linear(64→5)
  Total params: 6,757  ← MOST COMPACT MODEL IN THE E-SERIES

Loss   : MultiLabelSoftMarginLoss (train mask only)
Setting: Transductive (all nodes in forward pass; loss on train mask)
Optim  : Adam (lr=1e-3, wd=5e-4), CosineAnnealingLR (200 ep)
Threshold: 0.5

Train mask: 16724 nodes (80%)
Test mask:   4182 nodes (20%)

TRAINING DYNAMICS

Epochs 1-20: Macro-F1 fluctuates (0.03-0.15) — GCN calibrating neighbour
  feature aggregation, initially harmed by noisy neighbourhood labels.
Epochs 20-100: Steady improvement 0.05 → 0.41 (consistent learning)
Epochs 100-200: Continued slow rise 0.41 → 0.47 (NOT converged at ep200)

⚠️ KEY OBSERVATION: Loss still decreasing at epoch 200 (train=0.4915,
val=0.4910). The model has NOT converged. More epochs (e.g., 400-500)
would very likely push Macro-F1 toward 0.50+ range and AUPRC above 0.62.

The slow convergence is typical of GCN in transductive settings — the graph
structure takes many propagation steps to diffuse informative features from
train nodes to their test node neighbours.

=====================================================================
KEY INTERPRETATIONS
=====================================================================

1. BEST AUPRC (0.6023) — GRAPH ADDS CALIBRATION VALUE:
   Graph neighbourhood aggregation smooths probability estimates by averaging
   over similar clips. This reduces overconfident predictions and produces
   better-calibrated scores. Highest AUPRC means E1's logit rankings are the
   most trustworthy for threshold-agnostic evaluation (clinical triage).

2. HIGHEST PRECISION (0.675) — MOST CONSERVATIVE POSITIVE PREDICTIONS:
   GCN only predicts positives when neighbours also agree. This consensus
   mechanism dramatically reduces false positives but misses isolated
   stutter events. Precision=0.675 is the highest of any model.

3. INTERJECTION IS STRONGEST (F1=0.605, AUPRC=0.723):
   Interjection clips likely cluster tightly in PCA space (similar filler
   words: "um", "uh") making their graph neighbourhood highly informative.
   k-NN neighbours of an Interjection clip are likely also Interjection —
   so GCN correctly propagates this signal.

4. WORDREP IS WEAKEST (F1=0.284, AUPRC=0.464):
   WordRep clips are heterogeneous (any repeated word) and sparse (18%).
   Their k-NN neighbours may be any stutter type, diluting the signal.
   This is consistently the weakest class across all experiments.

5. STILL CONVERGING — MORE EPOCHS WOULD HELP:
   The monotonic increase from ep10→ep200 with no sign of plateau
   suggests running 400+ epochs would materially improve Macro-F1.
   This is noted for future work but not re-run in this session.

6. 6,757 PARAMETERS — MOST COMPACT COMPETITIVE MODEL:
   The GCN is the smallest model in the experiment suite:
   E1: 6,757 params vs F1: 174K vs D1: 1.56M
   The graph structure provides the "compute" — not the model itself.
