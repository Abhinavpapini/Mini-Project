MODEL CONFIGURATION
=====================================================================

Feature input:
  - HuBERT-large layer 21 (1024-dim → PCA-32, expl. variance 58.3%)
  - MFCC-21 (first 21 coefficients from 78-dim stats cache, standardised)
  - Fused dim: 53  (32 + 21)

Model architecture: Conformer multi-label encoder
  - Input: [B, 53] → unsqueeze → [B, 53, 1] → Linear(1→64) → [B, 53, 64]
  - Each of 53 feature positions treated as a 1D "token"
  - 2× ConformerBlock:
      FF(half-step) → MHSA(4 heads, d=64) → DepthwiseConv(k=7) → FF(half-step) → LN
  - AdaptiveAvgPool over 53 positions → [B, 64]
  - LN → Dropout(0.1) → Linear(64→128) → GELU → Dropout(0.1) → Linear(128→5)
  - Total params: 202,373

=====================================================================
KEY INTERPRETATIONS
=====================================================================

1. CONFORMER ≈ CNN FOR THIS TASK: The Conformer's self-attention over 53 feature
   positions yields nearly identical results to the CNN-1D in F1. With only 53
   tokens of dim=1 (projected to 64), there is limited sequence structure for
   attention to exploit. Conformer advantage likely requires longer input sequences.

2. BCE vs FOCAL LOSS: F1's focal loss (gamma=2) marginally outperforms F4's BCE
   on Macro-AUPRC (0.581 vs 0.561), meaning focal loss produces better-calibrated
   probability rankings. BCE's advantage is simpler tuning; focal's advantage is
   handling extreme class imbalance better.

3. HIGH RECALL PATTERN PERSISTS: Both F1 and F4 show recall >>  precision (0.725
   vs 0.452). This is a consistent signature of the class-balanced sampler — it
   aggressively up-weights rare stutter clips, pushing the model to classify
   positively, which boosts recall but hurts precision.

4. WORD REPETITION CONSISTENTLY HARDEST: WordRep F1 = 0.421 (F4), 0.415 (F1).
   This class needs full temporal sequence modeling (D-series) rather than
   pooled features. Cannot be solved by loss function choice alone.

5. TRAINING IS SLOW TO CONVERGE: F4 macro_f1 only reaches 0.42-0.47 in epochs 1-6,
   then steadily climbs. The Conformer requires more warmup due to attention
   initialization. Could benefit from warmup LR schedule + 50 epochs.

Key Hypotheses and Expected Outcomes
------------------------------------
- H1: Layer optimality is stutter-type dependent: Blocks (motor planning failure)
  may be better captured in later layers encoding prosodic structure, while sound
  repetitions (acoustic-phonetic events) may peak in mid-layers. Expected: per-type
  optimal layers differ by 3 to 8 layers.
- H2: Dimensionality reduction improves generalization: PCA to about 32 dimensions
  from 1024 should improve macro-F1 on minority stutter types by removing noise
  dimensions, similar to the 32-dim result in the dysarthria paper.
- H3: Handcrafted stutter features provide complementary signal: Features like
  jitter, pause ratio, and speaking rate should show low correlation with compact
  SSL+MFCC fusion (similar to <0.1 mean correlation in the dysarthria paper),
  indicating additive value.
- H4: Graph-based models outperform CNN on temporal stutter patterns: Disfluencies
  span multiple frames and involve non-local acoustic dependencies (for example,
  coarticulation changes before a block). GCN/D-GAT are expected to capture these
  better than frame-averaged CNN classifiers.
- H5: Multi-label formulation improves co-occurrence handling: Converting to
  multi-label binary classification should improve F1 on clips with two or more
  simultaneous disfluency annotations, expected to be a significant portion of
  SEP-28k.
