=====================================================================
WHY TEMPORAL EDGES DID NOT HELP
=====================================================================

1. STUTTER TYPE IS NOT TEMPORALLY CORRELATED WITHIN EPISODES:
   Adjacent clips in the same podcast episode can have ANY stutter type.
   A Block event at clip_i doesn't predict Prolongation at clip_{i+1}.
   The temporal ordering captures conversational flow, not disfluency type.

2. λ=0.5 TEMPORAL WEIGHTING INTRODUCES NOISY SIGNAL:
   20,648 temporal edges carry diluted information. While not as harmful
   as E3's coarse hyperedges (mean 209 nodes), temporal neighbours of
   different stutter types still add noise to the propagated representation.

3. SPATIAL K-NN ALREADY ENCODES THE KEY SIMILARITY:
   The 162,328 k-NN spatial edges group acoustically similar clips — which
   implicitly captures some temporal autocorrelation (same speaker in the
   same session produces similar speech). Adding explicit temporal links is
   partially redundant AND adds noisy cross-stutter-type connections.

4. CLIP ORDERING NOT STRICTLY TEMPORAL:
   Clip IDs within an episode don't perfectly guarantee strict time ordering
   (can depend on dataset construction). Some "temporal" edges may connect
   clips that are not truly adjacent in time, injecting random noise.

MODEL CONFIGURATION

Features: HuBERT-large layer 21, PCA-32 (expl_var=0.583)
Spatial edges: k=10 cosine nearest neighbours (162,328 undirected)
Temporal edges: within-(show,episode) sequential connections (20,648 undirected)
Combined adjacency: A = A_spatial + 0.5 * A_temporal, then D^(-1/2) normalise
Architecture: same 2-layer GCN as E1 (32→64→32→head, 6,757 params)
Loss: MultiLabelSoftMarginLoss (train mask), Adam (lr=1e-3, wd=5e-4)
Training: 200 epochs full-batch (train mask: 16724 nodes)

Key Hypotheses and Expected Outcomes
------------------------------------
- H1: Layer optimality is stutter-type dependent: Blocks (motor planning failure)
   may be better captured in later layers encoding prosodic structure, while sound
   repetitions (acoustic-phonetic events) may peak in mid-layers. Expected: per-type
   optimal layers differ by 3 to 8 layers.
- H2: Dimensionality reduction improves generalization: PCA to about 32 dimensions
   from 1024 should improve macro-F1 on minority stutter types by removing noise
   dimensions, similar to the 32-dim result in the dysarthria paper.
- H3: Handcrafted stutter features provide complementary signal: Features like
   jitter, pause ratio, and speaking rate should show low correlation with compact
   SSL+MFCC fusion (similar to <0.1 mean correlation in the dysarthria paper),
   indicating additive value.
- H4: Graph-based models outperform CNN on temporal stutter patterns: Disfluencies
   span multiple frames and involve non-local acoustic dependencies (for example,
   coarticulation changes before a block). GCN/D-GAT are expected to capture these
   better than frame-averaged CNN classifiers.
- H5: Multi-label formulation improves co-occurrence handling: Converting to
   multi-label binary classification should improve F1 on clips with two or more
   simultaneous disfluency annotations, expected to be a significant portion of
   SEP-28k.
