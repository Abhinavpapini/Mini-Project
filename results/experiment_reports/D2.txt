Key insight: PCA-01 gets MORE attention than PCA-00, despite PCA-00 having
nearly DOUBLE the explained variance (12.4% vs 7.0%). This means:
  - The highest-variance direction captures vocal energy/amplitude variation
    (speaker-level factor, not stutter-specific)
  - PCA-01 and PCA-02 capture finer phonetic variation that better
    discriminates stutter types
  - PCA dims 5-63 receive negligible attention (attn < 0.04 each)

This attention pattern is consistent with the finding that PCA-32 (D1)
is nearly as good as PCA-64 (D2) — the useful information is concentrated
in the top 4-5 components. 64 dims adds noise, not signal.

=====================================================================
MODEL CONFIGURATION
=====================================================================

Feature input:
  - HuBERT-large layer 21 (1024-dim → PCA-64)
  - Explained variance retained: 69.6% (vs D1's 58.3% at PCA-32)
  - 64 feature positions treated as a sequence

Architecture: D2 BiLSTM + Additive Attention Pool
  Token embedding: Linear(1→128) + LayerNorm + GELU  [per PCA position]
  BiLSTM: 2 layers, hidden=128, bidirectional
    → Output: [B, 64, 256]  (256 = 128 × 2 directions)
  Additive attention (Bahdanau):
    proj = tanh(Linear(256→64)) → score = Linear(64→1) → softmax
    context = Σ attn_weight × hidden_state  [B, 256]
  Head: LayerNorm → Dropout → Linear(256→128) → GELU → Dropout → Linear(128→5)
  Total params: 710,533

Loss: MultiLabelSoftMarginLoss (same as D1, D4, D5)
Sampler: WeightedRandomSampler (same cross-D-series)
Optimizer: AdamW (lr=3e-4, wd=1e-4), CosineAnnealingLR (30 ep)

=====================================================================
KEY INTERPRETATIONS
=====================================================================

1. BILSTM SEQUENTIAL PROCESSING INFERIOR TO CONFORMER CONV:
   PCA components are not a natural time sequence — treating them as one
   is a weaker inductive bias than D1's local conv (which captures adjacent
   co-variance). BiLSTM's recurrent processing cannot fully exploit PCA
   component ordering.

2. ATTENTION IS HIGHLY CONCENTRATED (TOP-5 DOMINATE):
   PCA dims 0-4 capture 86.7% of mean attention despite being 7.8% of the
   64 possible positions. This validates D1's choice of PCA-32 — the
   additional dims 32-63 are essentially ignored by the model.

3. PCA-01 > PCA-00 IN ATTENTION (INTERPRETABILITY FINDING):
   The model deprioritises the maximum-variance component (PCA-00), which
   captures broad prosodic/speaker variation. Instead it focuses on PCA-01/02
   which capture phoneme-level variation more specific to stutter events.
   This is a useful finding for feature selection in future work.

4. D2 NOT PROMOTED:
   Optional experiment, confirmed as weaker than D1. BiLSTM+AttnPool
   provides interpretability value but not prediction accuracy.
