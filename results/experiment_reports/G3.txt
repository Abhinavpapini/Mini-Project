Objective
---------
Implement the layer-wise probing + Gumbel-Softmax layer selection pipeline
(inspired by Batra et al., EUSIPCO 2025), adapted to SEP-28k only (no FluencyBank).

Pipeline:
  Stage 1 — LR probing across all cached layers per SSL model
             → identify best layers + initialize layer weights
  Stage 2 — WeightedLayerSum (learnable α from probe scores)
             → CNN-1D over model streams
             → GumbelClassRouter (per-class, τ annealed 2.0→0.5)
             → 6-class single-label softmax head

Task Adaptation: SEP-28k multi-label → 6-class single-label
  Priority: Block > Interjection > Prolongation > SoundRep > WordRep > Fluent
  Classes: Block | Prolongation | SoundRep | WordRep | Interjection | Fluent

KEY FINDINGS
-------------------------------------------------------------------
1. GUMBEL ROUTING IS SEMANTICALLY MEANINGFUL:
   The model independently learned the same HuBERT/Whisper split that
   prior experiments (C4, D7) found manually:
   - HuBERT → acoustic/phonetic stutter types (Block, Prolongation, SoundRep)
   - Whisper → linguistic/boundary types (WordRep, Interjection, Fluent)
   This validates the complementarity hypothesis from scratch via gradient learning.

2. LAYER 21 (HuBERT) AND L31 (Whisper) REMAIN CANONICAL:
   Stage 1 probing on the 6-class task confirms the same optimal layers
   found in all prior multi-label experiments. Layer selection is robust
   across task formulations.

3. MINORITY CLASSES REMAIN THE BOTTLENECK:
   SoundRep F1=0.156 and WordRep F1=0.273 are significantly worse than
   majority classes. With only 722 and 703 samples respectively, even
   weighted sampling cannot fully compensate. A focal loss or data
   augmentation strategy is needed for these types.

4. UNIFORM LAYER WEIGHTS = MORE LAYERS NEEDED:
   The near-uniform α weights suggest the model cannot clearly distinguish
   which of the 5 top layers is best — likely because the layer differences
   are small within the top-5 range. A wider sweep (e.g., all 24 HuBERT layers)
   would give more diverse layer choices and sharper learned weights.

5. MODEL IS NOT FULLY CONVERGED AT EPOCH 40:
   val Macro-F1 was still slowly increasing at epoch 40 (0.383 → 0.385).
   Running 60-80 epochs would likely yield ~0.40+ Macro-F1.
