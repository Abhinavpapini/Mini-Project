Objective
---------
Fuse Whisper-large-v3 encoder representations (linguistic/ASR context) with
HuBERT-large representations (acoustic structure) for stutter type classification.
Hypothesis: Whisper captures linguistic disfluency context; HuBERT captures acoustic
phonatory structure — combining them should improve over single-model baselines.

Execution command used
----------------------
.venv\Scripts\python.exe experiments/C5_whisper_hubert_bilstm.py \
    --hubert-alias hubert-large --hubert-layer 21 \
    --whisper-alias whisper-large --whisper-layer 24 \
    --target Block --fold fold0 \
    --ssl-pca-dim 32 --proj-dim 64 \
    --lstm-hidden 128 --lstm-layers 2 \
    --epochs 20 --batch-size 256 --lr 3e-4 --dropout 0.3

Interpretation
--------------
1. C5 F1 = 0.6311 is a clear improvement over:
   - C1 (SSL+MFCC)           : 0.5268  (+0.1043)
   - C2 (SSL+handcrafted)    : 0.5550  (+0.0761)
   - C3 (multi-SSL Transformer): 0.5674 (+0.0637)
   This makes C5 the best fusion result so far on the Block target.

2. Stream attention is nearly perfectly equal (50.06% vs 49.94%), indicating
   both HuBERT acoustic features and Whisper linguistic features contribute
   equally to Block classification. This is consistent with Block being both
   a motor-phonatory event (HuBERT strength) and a prosodically-marked
   disfluency (Whisper strength).

3. Best epoch was early (epoch 4 out of 20), with val_f1 plateauing around
   0.62-0.63 afterward. This suggests the model converges quickly and that
   the PCA-32 bottleneck may be limiting further gains. Possible improvement:
   increase PCA dim to 64 or 128 for remaining 4 stutter types.

4. Precision (0.5747) < Recall (0.6997): The model has more false positives
   than false negatives — it is biased toward predicting Block. The class-
   weighted loss partially corrects this but Block's 42.45% base rate still
   challenges precision.

5. Training loss continued decreasing through epoch 20 while val loss plateaued
   after epoch 7, indicating mild overfitting. Consider adding early stopping
   or increasing dropout to 0.4 for other targets.
